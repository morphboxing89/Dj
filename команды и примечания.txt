django-admin - список доступных команд

django-admin startproject app . - создаем приложение(название можно давать любое)
создаст папку с проектом и файл manage

Чтобы запустить приложение как обычную программу - возле дебаггера жмем ... переходим в edit
выбираем путь к файлу, интерпритаторо и доп параметр указываем runsrver

python manage.py startapp users - создаст папку с  приложением

после создания приложения в файле settings -> INSTALLED_APPS, прописываем название приложения, тем самым
регестрируя его

__init__ - сообщает python, что файлы связаны между собой
admin.py - нужен чтобы мы зарегестрировали таблицы которые будут созданы в админ панеле и настроили их отображение
в админ панеле по управлению веб-приложением для обслуживающего персонала
apps.py - настройка и конфигурация приложения
models - срздаются спец классы которые описывают какие таблицы и какая информация в них будет храниться
tests - для тестов
В views - прописываются функции запросов от пользователя

Шаблонные теги
    {% if %}
    {% endif %}

{% load static %} - подгружает в шаблон маршруты по которым django будет искать папку со статическими файлами

в файле settings STATICFILES_DIRS - указываем дополнительные места поиска статики
